# Edvoy Genie - Premium UI Pro

A high-performance, aesthetically pleasing chat interface for the Edvoy Study Abroad Assistant.

## Features
- **Modern Design**: Deep dark theme with glassmorphism and smooth animations.
- **Real-time Streaming**: Connects directly to the Edvoy Genie API for instant, line-by-line response rendering.
- **Dynamic Status Updates**: Visual indicators for AI "Thinking" and "Searching" states.
- **Responsive Layout**: Works beautifully on desktops and tablets.

## Getting Started

### 1. Install Dependencies
Navigate to this directory and install the necessary development tools:
```bash
npm install
```

### 2. Run in Development Mode
Start the local development server:
```bash
npm run dev
```

### 3. Build for Production
Create an optimized production bundle:
```bash
npm run build
```

## Configuration
The UI is currently configured to connect to the backend at `http://localhost:4110`. You can change this in `src/main.js` if your API is hosted elsewhere.
